# 🌍 Around The World

**Around The World** is a modern React application that allows users to explore countries from all over the globe.  
It provides detailed information about each country, including population, region, capital, and neighboring countries, all wrapped in a clean, responsive, and user-friendly interface.

🚀 **Live Demo:**  
https://around-the-world-orpin.vercel.app/

📂 **GitHub Repository:**  
https://github.com/AbdullahSameh10/Around-The-World

---

## ✨ Features

- 🌎 Browse all countries worldwide  
- 🔍 Search for a country by name  
- 🗺️ Filter countries by region  
- 📄 View detailed country information  
- 🤝 Display bordering (neighbor) countries  
- 💾 Local storage caching for better performance  
- ⏳ Loading & error handling states  
- 📱 Fully responsive design  

---

## ⚙️ Installation & Setup

To run this project locally, follow these steps:

1- Clone the repository
```bash
git clone https://github.com/AbdullahSameh10/Around-The-World.git
```

2- Navigate to the project folder
```bash
cd Around-The-World
```

3- Install dependencies
```bash
pnpm install
```

4- Start the development server
```bash
pnpm dev
```
### or
```bash
pnpm run dev
```

The app will run locally on:
```bash
http://localhost:5173
```
